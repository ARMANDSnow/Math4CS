## 二、A 老师 · 计算大题"标准模板"

### 模板 1 · LU 分解解 $Ax=b$
1. 前向消元：乘子 $m_{ij}=a_{ij}/a_{jj}$，把对角线下方消成 0 → 得上三角 $U$。
2. $L$ = 下三角，对角线全 1，下方填**正的乘子** $m_{ij}$。
3. 前代 $Ly=b$ 求 $y$；回代 $Ux=y$ 求 $x$。
4. 代回 $Ax$ 验算。
> 口诀：**消元写 U → 乘子写 L → 前代 y → 回代 x**。

### 模板 2 · 最小二乘 / 参数回归（大题热点）
- 建模：$\min_x\|Ax-b\|_2^2$（预测 $Ax$ 逼近观测 $b$）。
- **正规方程**：$A^\top Ax=A^\top b\Rightarrow x=(A^\top A)^{-1}A^\top b$。
- **加 Tikhonov**：$\min\|Ax-b\|^2+\lambda\|x\|^2\Rightarrow(A^\top A+\lambda I)x=A^\top b$。
- 欠定最小范数：$x=A^\top(AA^\top)^{-1}b$。
> 展开式必记：$\|Ax-b\|^2=x^\top A^\top Ax-2b^\top Ax+b^\top b$。

### 模板 3 · QR 分解
**Gram-Schmidt**：
1. $q_1=a_1/\|a_1\|$；
2. $v_j=a_j-\sum_{i<j}(a_j^\top q_i)q_i$，$q_j=v_j/\|v_j\|$；
3. $R_{ij}=q_i^\top a_j$（上三角），$R_{jj}=\|v_j\|$。

**Householder**：
1. 对每列取 $v=x\mp\|x\|e_1$，构造 $H=I-\dfrac{2vv^\top}{v^\top v}$（对称正交）；
2. 逐列把对角线下方清零 → $R=H_{n-1}\cdots H_1A$，$Q=H_1\cdots H_{n-1}$。
> 验收：$Q$ 列正交归一、$R$ 上三角、$QR=A$ 即给分（符号差异不扣分）。

### 模板 4 · SVD 计算
1. 算 $A^\top A$，特征分解：特征值 $\lambda_i$、特征向量 = $V$ 的列；
2. 奇异值 $\sigma_i=\sqrt{\lambda_i}$（降序），$\Sigma$ 放对角；
3. $u_i=Av_i/\sigma_i$ → 组成 $U$。
> 验收：$U,V$ 正交、$\Sigma$ 降序非负、$U\Sigma V^\top=A$。$\|A\|_2=\sigma_1$，$\kappa_2=\sigma_{\max}/\sigma_{\min}$。

### 模板 5 · 牛顿法 / 割线法
- **牛顿**：$x_{k+1}=x_k-\dfrac{f(x_k)}{f'(x_k)}$（二次收敛，需 $f'$）。
- **割线**：$x_{k+1}=x_k-f(x_k)\dfrac{x_k-x_{k-1}}{f(x_k)-f(x_{k-1})}$（超线性，免 $f'$，要两初值）。
- 从 $x_0,x_1$ 起，"迭代 $k$ 次"得到 $x_2,\dots,x_{k+1}$；每步代回算 $f$。

## 三、A 老师 · 必背概念速答

| 考点 | 标准答案 |
|---|---|
| 前向误差 | $\lvert x_{\text{est}}-x^\star\rvert$（需真根，通常算不出） |
| 后向误差（残差） | $\lvert f(x_{\text{est}})\rvert$（可算） |
| 条件数 | 前向/后向；根处 $\approx 1/\lvert f'(x^\star)\rvert$；求极小用 $1/\lvert f''(x^\star)\rvert$ |
| 矩阵范数 | $\|A\|_1$=最大列和、$\|A\|_\infty$=最大行和、$\|A\|_2=\sigma_{\max}$ |
| 矩阵条件数 | $\kappa(A)=\|A\|\|A^{-1}\|$；$\kappa_2=\sigma_{\max}/\sigma_{\min}$；$\kappa_2(A^\top A)=\kappa_2(A)^2$ |
| Cholesky | SPD 的 $A=LL^\top$（不考计算）；与 QR 关系 $L=R^\top$ |
| 谱定理 | 对称阵：实特征值、正交特征向量、$A=Q\Lambda Q^\top$ |
| 幂迭代 | 找最大模特征值；收敛率 $\lvert\lambda_2/\lambda_1\rvert$；两最大模相等时**不收敛** |
| 伪逆 | $A^+=V\Sigma^+U^\top$，统一解最小二乘/最小范数 |
| Eckart-Young | 最佳秩 $k$ 近似 = 取前 $k$ 个奇异值，误差 $\sigma_{k+1}$ |
| 方法选择 | $f'$易→牛顿；$f'$难→割线；Lipschitz $c<1$→不动点；连续不可微→二分 |

## 四、B 老师 · 必拿分

### 贝叶斯网络三件套（旧 Q4，13 分）
1. **联合分解**：$P(X_1,\dots,X_n)=\prod_i P(X_i\mid\text{Parents}(X_i))$。
2. **条件独立（d-分离）**：链/分叉 → 观测中间点阻断；汇聚 → 观测汇聚点反而打开。两点间所有路径阻断 ⇒ 独立。
3. **马尔科夫毯**：父 ∪ 子 ∪ 子的其他父。
4. **后验**：$P(D{=}1\mid S{=}1)=\dfrac{P(S{=}1\mid D{=}1)P(D{=}1)}{\sum_{d}P(S{=}1\mid D{=}d)P(D{=}d)}$。

### 信息论计算（旧 Q3 + HW2）
- $H(X)=-\sum p\log_2 p$；$H(Y\mid X)=H(X,Y)-H(X)$；$I(X;Y)=H(X)+H(Y)-H(X,Y)$。
- 交叉熵 $\text{CE}(p,q)=-\sum p\log_2 q$；$D_{\mathrm{KL}}=\text{CE}-H$。
- 决策树选特征：信息增益 $I(Y;A)$ 最大者。

### 优化（旧 Q5）
- 问题分类：目标/约束 线性→LP、凸二次→QP、凸二次约束→QCQP、posynomial→GP。
- Lagrange：$L(x,\lambda)=f_0(x)+\sum\lambda_i g_i(x)$；对偶函数 $g(\lambda)=\min_x L(x,\lambda)$。

### Q3（判断题，10 分）
| 小题 | 判断 | 理由 |
|---|---|---|
| (1) 贝叶斯学派概率随经验更新 | **对** | 贝叶斯派把概率看作主观信念、随证据更新 |
| (2) 熵越大对事情越确信 | **错** | 熵越大 = 越**不确定**，不是确信 |
| (3) 交叉熵表示两分布的距离 | **对** | 交叉熵/KL 衡量分布差异 |
| (4) 牛顿法用二阶泰勒、比梯度下降快 | **对** | 优化里牛顿用 Hessian（二阶），局部收敛更快 |
| (5) ILP 是对 LP 的近似 | **错** | 反了——LP 松弛才是 ILP 的近似；ILP 更难（NP-hard） |

### Q4（贝叶斯网络，13 分）
网络：$D_1\to S_1$；$D_1,D_2\to S_2$；$D_1,D_3\to S_3$；$D_3\to S_4$。
- **(1) 已知 $D_1$，$S_1\perp S_2$？** 唯一路径 $S_1\leftarrow D_1\to S_2$ 是分叉，$D_1$ 已观测 → 阻断 → **独立**。
- **(2) $S_2$ 马尔科夫毯**：父 $\{D_1,D_2\}$，无子节点 → $\text{MB}(S_2)=\{D_1,D_2\}$。
- **(3) $P(D_3{=}1\mid S_4{=}1)$**：$=\dfrac{0.9\times0.1}{0.9\times0.1+0.6\times0.9}=\dfrac{0.09}{0.63}\approx0.143$。

### Q5（优化，10 分）
$\min\ \tfrac12(x_1^2+x_2^2)$ s.t. $2x_1-x_2=5$。
- **(1) 类型**：目标凸二次（Hessian $=I\succ0$）+ 线性等式约束 → **凸二次规划 QP**。
- **(2) Lagrange 对偶**：$L=\tfrac12(x_1^2+x_2^2)+\lambda(2x_1-x_2-5)$。
  令 $\partial_x L=0$：$x_1=-2\lambda,\ x_2=\lambda$。代回：
  $$g(\lambda)=-\tfrac52\lambda^2-5\lambda$$

### Q6（误差分析，8 分）
求 $f(x)=0$ 的根：
- (1) 前向误差 $\lvert x_{\text{est}}-x^\star\rvert$；
- (2) 后向误差 $\lvert f(x_{\text{est}})\rvert$；
- (3) 条件数 $\dfrac{\lvert x_{\text{est}}-x^\star\rvert}{\lvert f(x_{\text{est}})\rvert}$；
- (4) $x^\star$ 附近 $\approx\dfrac{1}{\lvert f'(x^\star)\rvert}$（泰勒：$f(x_{\text{est}})\approx f'(x^\star)(x_{\text{est}}-x^\star)$）。

### Q7（线性系统，18 分）
- **(1) 图像去模糊**：(i) $\min_x\|Gx-x_0\|_2^2$；(ii) 加 Tikhonov $\min_x\|Gx-x_0\|_2^2+\lambda\|x\|_2^2$。
- **(2) Cholesky-QR**：(i) $M=A(L^\top)^{-1}$，$M^\top M=L^{-1}(LL^\top)(L^\top)^{-1}=I$ → 正交；(ii) $A=QR\Rightarrow A^\top A=R^\top R=LL^\top\Rightarrow L=R^\top$。
- **(3) SVD**：(i) $U_{m\times m}$、$V_{n\times n}$ 正交，$\Sigma_{m\times n}$ 对角（奇异值降序非负），$\sigma_i=\sqrt{\lambda_i(A^\top A)}$，rank = 非零奇异值数；(ii) $V^\top$ 旋转 → $\Sigma$ 沿轴拉伸 → $U$ 旋转。

### Q8（非线性求根方法选择，8 分）
| 条件 | 方法 |
|---|---|
| $f\in C^1$，$f'$ 易算 | 牛顿法 |
| $f\in C^1$，$f'$ 难算 | 割线法 |
| Lipschitz $0\le c<1$ | 不动点迭代 |
| 连续但不可微 | 二分法 |

## 六、终极速记（公式 + 口诀）

**数值线代主线**：误差/条件数 → LU 解方阵 → 最小二乘(正规方程) → 范数/条件数 → QR(去 $A^\top A$) → 特征值 → SVD(集大成) → 非线性求根。

**核心公式串**：
- 正规方程 $A^\top Ax=A^\top b$；Tikhonov $(A^\top A+\lambda I)x=A^\top b$；最小范数 $A^\top(AA^\top)^{-1}b$。
- $\|A\|_1$列、$\|A\|_\infty$行、$\|A\|_2=\sigma_1$；$\kappa_2=\sigma_{\max}/\sigma_{\min}$。
- QR：$A=QR$，$Rx=Q^\top b$。SVD：$A=U\Sigma V^\top$，$A^+=V\Sigma^+U^\top$。
- 牛顿 $x-f/f'$；割线 $x-f\cdot\frac{\Delta x}{\Delta f}$。
- 贝叶斯 $\frac{P(B|A)P(A)}{\sum P(B|A_i)P(A_i)}$；熵 $-\sum p\log_2 p$；$I=H(X)+H(Y)-H(X,Y)$。
- 马尔可夫 $\pi_t=\pi_0P^t$，$(P^n)_{ij}$；平稳 $\pi=\pi P$。

**口诀**：
- 误差："前向看解、后向代回方程"；导数小→条件数大→病态。
- LU："消元 U、乘子 L、前代后代"。
- QR："去投影归一化"（GS）/"镜面反射清零"（Householder）。
- SVD："$A^\top A$ 特征 → $\sigma$ 开方 → $u=Av/\sigma$"。
- 求根："导数易牛顿、难算割线、压缩不动点、不可微二分"。
- 贝叶斯网："链叉观测断、汇聚观测通；马毯=父+子+子的父"。
- 信息论："熵=不确定，越大越难猜"。
