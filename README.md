# Math4CS 复习讲义

上海交通大学《计算机科学的数学基础》(EST2324 / SE2324) 期末自学讲义合集。本课由两位老师合上、期末合卷考，本仓库覆盖两部分全部考点。

## 内容

### 应凯老师 · Numerical Algorithms（数值算法）
- [NA1 数值与误差分析](NA1-数值与误差分析.md) — 前向/后向误差、条件数
- [NA2 线性系统与 LU 分解](NA2-线性系统与LU分解.md) — 高斯消元、LU
- [NA3 最小二乘与参数回归](NA3-最小二乘与参数回归.md) — 正规方程、Tikhonov、Cholesky
- [NA4 范数与条件数](NA4-范数与条件数.md) — 向量/矩阵范数、κ(A)
- [NA5 列空间与 QR](NA5-列空间与QR.md) — Gram-Schmidt、Householder
- [NA6 特征值问题](NA6-特征值问题.md) — 谱定理、幂迭代
- [NA7 奇异值分解 SVD](NA7-奇异值分解SVD.md) — 伪逆、低秩近似
- [NA8 非线性方程求根](NA8-非线性方程求根.md) — 牛顿法、割线法、二分、不动点

- [Ch1 概率与贝叶斯网络](Ch1-概率与贝叶斯网络.md)

### 导航与冲刺
- [00 考纲与考点分布](00-考纲与考点分布.md) — 总导航、真题对照、复习优先级
- [考前重点突破](考前重点突破.md) — 解题模板 + 旧真题全解 + 口诀

## 体例

每章 8 段：解决什么问题 → 从零铺垫 → 核心概念 → 公式推导 → 手算例题 → 考点雷达 → 易错点 → 速查卡。例题取自课程作业与往年真题，所有手算数值均经程序核验。
